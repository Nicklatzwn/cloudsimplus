org.cloudbus.cloudsim.schedulers.cloudlet.network
=================================================

Provides \ :java:ref:`org.cloudbus.cloudsim.schedulers.cloudlet.network.CloudletTaskScheduler`\  implementations to perform network packet dispatching by a regular \ :java:ref:`org.cloudbus.cloudsim.schedulers.cloudlet.CloudletScheduler`\ . Such PacketSchedulers are attached to a CloudletScheduler to receive/send packets from/to a Cloudlet.
